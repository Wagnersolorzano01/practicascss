Evolución de las Versiones de HTML

A lo largo del tiempo, HTML ha evolucionado incorporando nuevas funcionalidades para mejorar la estructura, el diseño y la experiencia del usuario.

HTML 1.0 (1991)

HTML 1.0 fue la primera versión creada por Tim Berners-Lee. Permitía documentos sencillos con encabezados, párrafos, listas y enlaces.

  • Uso básico de etiquetas
  • Estructura simple
  • Sin soporte multimedia

HTML 2.0 (1995)

HTML 2.0 fue la primera versión estandarizada, incorporando formularios y tablas.

  • Estandarización oficial
  • Soporte para formularios
  • Mejor organización del contenido

HTML 3.2 (1997)

Introdujo mejoras visuales, tablas avanzadas y soporte inicial para diseño.

  • Tablas más completas
  • Texto alrededor de imágenes
  • Elementos de formulario mejorados

HTML 4.01 (1999)

Se convirtió en el estándar principal durante muchos años, separando estructura y presentación mediante CSS.

  • Soporte completo para CSS
  • Mejor accesibilidad
  • Formularios avanzados

HTML5 (2008 – 2014)

HTML5 introduce etiquetas semánticas, multimedia nativa y soporte para aplicaciones web modernas.

  • Audio y video nativos
  • Etiquetas semánticas
  • Compatible con dispositivos móviles
